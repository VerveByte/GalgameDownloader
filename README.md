# GalgameDownloader
一个聚合了多个galgame资源网站的galgame搜索下载工具

作者是个初三生，空闲时间不多，所以目前加入的网站不多~~（其实只有1个）~~但这并不会影响太多。

以后会继续往里添加更多网站

## 使用方法
- 请先安装node.js
```powershell
npm install
node cli.js "游戏名称"
```
